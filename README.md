# База данных сотрудников организации

Программный комплекс на C++ для ведения кадрового учёта с возможностью
экспорта данных в сторонние организации.

## Возможности

- Добавление, редактирование и удаление записей о сотрудниках
- Поиск по имени и фильтрация по отделу
- Сохранение базы в собственный формат `.mydb`
- Загрузка базы из файла с защитой от повреждённых данных
- Экспорт в CSV для банков, налоговых органов и других организаций
- Корректная работа с русским языком (UTF-8)

## Как запустить

1. Скопировать файлы из репозитория
2. Открыть проект в Visual Studio
3. В настройках проекта убедиться, что установлен флаг `/utf-8`
4. Собрать решение (Ctrl+Shift+B) и запустить (Ctrl+F5)

## Структура проекта

| Файл | Назначение |:
| `zagolovky.h` | Заголовочный файл: классы Employee, Database, перечисления |
| `kursach1.cpp` | Реализация всех методов |
| `menu.cpp` | Консольный интерфейс пользователя |

## Форматы файлов

- `.mydb` — собственный формат хранения базы (текстовый, с разделителями)
- `.csv` — экспортный формат для сторонних организаций
